psych_gonogo2

Average performance and reaction time.

Description

psych_gonogo2(cellids,issave) calculates average performance and reaction time for all sessions corresponding to cellids, per animal. Sessions that contain light-stimulation are excluded.

Input parameters:

cellids - list of cell IDs or index set to CELLIDLIST (see CellBase documentation); if empty or not specified, all cells are selected from the CellBase

issave - controls saving

Plots:

1. Average performance and reaction time per mouse for the sessions in which the cells (cellids) were recoreded. The sound intensities are normalized between 0 and 1 within each session. After averaging according to normalized intensities, the intensities are evenly distributed for plotting.

2. In another within-mouse average, all intermediate intensities are pooled within a session. For plotting, these intensities are displayed in the middle.

3. Grand average: average across mice from the second type of within-mouse averages.
