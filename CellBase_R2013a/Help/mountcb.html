mountcb

Mount CellBase.

Description

mountcb mounts an existing CellBase in Matlab. The new CellBase is added to the Matlab CellBase preferences.

mountcb(name) mounts new CellBase with the specified name. Multiple instances of CellBase can be initialized with different names. Default CellBase can be set using choosecb.

See Also
